# Stock Trading Platform

Full-stack stock trading platform with React frontend and dashboard.

## Project Structure

- `frontend/` - Main trading platform UI
- `dashboard/` - Admin/Analytics dashboard  
- `backend/` - API server

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm start
```

### Dashboard
```bash
cd dashboard
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```
